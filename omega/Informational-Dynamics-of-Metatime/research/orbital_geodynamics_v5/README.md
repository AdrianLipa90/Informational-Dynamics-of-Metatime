# orbital_geodynamics_v5

V5 extends the global Euler-Berry-Poincare-421 + A_ij/tau_i model with:
- Monte Carlo over free parameters,
- spectral analysis of the complex coupling matrix A_ij,
- Laplacian/Fiedler diagnostics,
- parameter sensitivity ranking against a composite objective.

Outputs:
- `results/v5_monte_carlo_results.json`
- `results/v5_monte_carlo_summary.md`

Scope:
This is the first statistical phase-diagram layer for the orbital repository model.
