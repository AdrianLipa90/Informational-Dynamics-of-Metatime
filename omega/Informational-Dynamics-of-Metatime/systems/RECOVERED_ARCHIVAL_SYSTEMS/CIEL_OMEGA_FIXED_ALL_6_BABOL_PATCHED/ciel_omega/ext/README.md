# CIEL/Ω — ext/ (DUMP STACK)

**Ten folder NIE jest częścią kernela.**

Nic z `ext/` nie może być importowane przez kod poza `ext/`.

Zawartość:
- Oryginalne prototypy (ext1–ext21, extemot, extfwcku)
- Porównania z ZIP
- Tymczasowe demka

Cały kod został rozbity i przeniesiony do odpowiednich pakietów w `ciel_omega/`.
Te pliki zachowane jako archiwum referencyjne.
